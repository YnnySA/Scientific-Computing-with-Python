SOLUCIONADOR DE SUDOKU
=====================

Este proyecto es parte del curso "Scientific Computing with Python" de freeCodeCamp, diseñado para demostrar conceptos avanzados de programación en Python.

DESCRIPCIÓN
-----------
El solucionador de Sudoku es una implementación que resuelve automáticamente puzzles de Sudoku utilizando el algoritmo de backtracking. El programa toma un tablero 9x9 parcialmente completo y encuentra la solución que cumple con todas las reglas del Sudoku:

- Cada fila debe contener los números del 1 al 9 sin repetición
- Cada columna debe contener los números del 1 al 9 sin repetición
- Cada cuadrado 3x3 debe contener los números del 1 al 9 sin repetición

CARACTERÍSTICAS
-------------
- Visualización clara del tablero usando asteriscos (*) para celdas vacías
- Implementación recursiva del algoritmo de backtracking
- Validación de movimientos en filas, columnas y cuadrados 3x3
- Interfaz simple para ingresar y resolver puzzles

FINALIDAD EDUCATIVA
------------------
Este proyecto demuestra:
- Programación orientada a objetos en Python
- Algoritmos recursivos
- Técnicas de backtracking
- Manipulación de matrices bidimensionales
- Métodos especiales de Python (__str__, __init__)
